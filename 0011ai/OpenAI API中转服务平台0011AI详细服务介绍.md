## 五、接入配置方法

### 1. API调用示例

使用标准的OpenAI接口协议，只需将官方接口地址替换为0011AI中转地址：

```python
import openai

# 配置0011.ai的中转地址和密钥
client = openai.OpenAI(
    base_url="https://api.0011.ai/v1",  # 0011.ai中转地址
    api_key="sk-xxxxxxxxxxxxxxxxxxxx"   # 您在平台获取的key
)

# 调用OpenAI Codex模型
response = client.chat.completions.create(
    model="gpt-5-codex",
    messages=[
        {"role": "system", "content": "你是一个资深软件工程师。"},
        {"role": "user", "content": "请重构这段Python代码以提高性能..."}
    ]
)
print(response.choices[0].message.content)
```

### 2. Codex CLI配置

在 `~/.codex/` 目录下创建 `config.toml` 文件：

```toml
model = "gpt-5-codex"
model_provider = "0011ai"
model_reasoning_effort = "medium"
disable_response_storage = true

[model_providers.0011ai]
name = "0011ai"
base_url = "https://api.0011.ai/v1"
wire_api = "responses"
```
