## 三、技术优势详解

### 3.1 中转服务架构

0011AI并非简单的API转发服务，而是采用了更先进的**账号池架构**：

1. **官方账号订阅**：平台批量订阅Anthropic和OpenAI的官方大额账号
2. **额度智能分配**：将官方额度分配给平台用户
3. **自动故障转移**：当某个账号被封禁时，后台自动切换至其他可用账号
4. **无感知切换**：用户端无需任何操作，服务持续可用

这种架构将封号风险完全转移给平台，用户可享受"零风险"使用体验。

### 3.2 支持的AI模型

平台支持多种Claude和Codex模型版本：

**Claude系列模型：**

- Claude Opus 4.5（最强性能）
- Claude Sonnet 4.5（性价比之选）
- Claude Haiku（轻量级快速响应）

**OpenAI Codex系列：**

- 支持代码生成、Bug修复、代码审查等功能
- 上下文记忆能力，支持多轮对话
- 兼容12+主流编程语言

## 五、注册与使用流程

### 5.1 注册步骤

1. **访问官网**：打开<https://0011.ai/i/PPT>
2. **账号注册**：点击右上角"注册"，输入邮箱并设置密码
3. **邮箱验证**：完成验证激活账号
4. **选择套餐**：根据需求选择按量付费或订阅会员
5. **支付充值**：使用微信扫码完成支付

### 5.2 Claude Code配置使用

**安装步骤**（无需提前安装Node.js）：

**Mac用户**：

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows用户**（管理员身份运行PowerShell）：

```powershell
irm https://claude.ai/install.ps1 | iex
```

**环境变量配置**：

```bash
# macOS/Linux
export ANTHROPIC_BASE_URL="https://aicoding.2233.ai"
export ANTHROPIC_API_KEY="你的API Key"
export ANTHROPIC_AUTH_TOKEN="你的API Key"
```

配置完成后，在终端输入`claude`即可启动使用。
