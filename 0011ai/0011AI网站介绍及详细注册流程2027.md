## 四、客户端配置与使用

### 一键安装Claude Code

**Mac用户**：

打开「终端」（Terminal），复制以下命令并执行：

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows用户**：

搜索「PowerShell」，右键选择「以管理员身份运行」，执行：

```powershell
irm https://claude.ai/install.ps1 | iex
```

安装完成后，使用以下命令查询版本确认安装成功：

```bash
claude --version
```

### API密钥配置

购买完成后，您可以在后台创建API密钥，然后根据平台提供的教程配置环境变量。将相关命令粘贴到终端执行即可。

### 启动使用

在终端输入以下命令启动Claude Code：

```bash
claude
```

中间会有确认提示，选择"Yes"继续。看到Claude标志即代表使用成功。
