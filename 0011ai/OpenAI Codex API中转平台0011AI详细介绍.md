将API基础地址指向0011AI中转平台：

**Python配置示例**：

```python
import os

# 配置客户端

    api_key=os.getenv("0011AI_API_KEY"),  # 从环境变量读取密钥
    base_url="https://api.0011ai.com/v1"   # 0011AI中转平台接入点
)
```

**Node.js配置示例**：

```javascript

const client = new OpenAI({
    apiKey: process.env.0011AI_API_KEY,
    baseURL: 'https://api.0011ai.com/v1'
});
```

### 4.3 Codex代码生成调用

Codex模型专门针对代码理解与生成优化，适用于以下任务：

```python
def generate_code(prompt, language="python"):
    """
    使用Codex模型生成代码
    """
    response = client.chat.completions.create(
        model="codex-latest",  # 或指定具体版本如codex-001
        messages=[
            {
                "role": "system",
                "content": f"你是一个专业的{language}程序员，只输出代码，不输出解释。"
            },
            {
                "role": "user",
                "content": prompt
            }
        ],
        temperature=0.2,  # 低温度确保代码确定性
        max_tokens=2000
    )
    return response.choices[0].message.content

# 使用示例
code = generate_code(
    "写一个Python函数，使用快速排序算法对列表进行排序，要求时间复杂度为O(n log n)",
    language="python"
)
print(code)
```
