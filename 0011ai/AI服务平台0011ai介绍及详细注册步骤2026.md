### 方式一：一键安装（推荐）

**Mac用户：**

1. 打开「终端」（Terminal）- 在启动台里搜索即可找到
2. 复制以下命令，粘贴进去，按回车：
  
  ```bash
  curl -fsSL https://claude.ai/install.sh | bash
  ```
  
3. 等待1-2分钟自动下载安装

**Windows用户：**

1. 搜索「PowerShell」，右键选择「以管理员身份运行」
2. 复制以下命令，粘贴进去，按回车：
  
  ```powershell
  irm https://claude.ai/install.ps1 | iex
  ```
  
3. 等待安装完成

### 方式二：使用0011ai CLI工具（可选）

```bash
# 安装0011.ai CLI工具

# 登录并配置密钥
0011ai login
0011ai config set api_key=your_api_key_here

# 启动Claude Code

```
