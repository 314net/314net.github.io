### 方式二：手动配置接入

如果你偏好手动配置，可以按照以下步骤操作：

#### 1. 安装 Claude Code CLI

```bash
# 使用 npm 安装 Claude Code
npm install -g @anthropic-ai/claude-code
```

#### 2. 配置 API 端点

编辑或创建配置文件（根据你的系统）：

**macOS/Linux**：

```bash
# 编辑 shell 配置文件
nano ~/.zshrc  # 或 ~/.bashrc

# 添加以下内容
export ANTHROPIC_API_KEY="sk-0011-你的密钥"
export ANTHROPIC_BASE_URL="https://api.0011.ai/v1"  # 0011.ai 代理端点
```

**Windows**：

```powershell
# PowerShell 中执行
[Environment]::SetEnvironmentVariable("ANTHROPIC_API_KEY", "sk-0011-你的密钥", "User")
[Environment]::SetEnvironmentVariable("ANTHROPIC_BASE_URL", "https://api.0011.ai/v1", "User")
```

#### 3. 验证连接

```bash
# 重新加载配置
source ~/.zshrc

# 启动 Claude Code
claude

# 看到欢迎界面即表示连接成功
```

## 七、常见问题解答（FAQ）

### Q1: 0011.ai 和官方 Claude Code 有什么区别？

**A**: 功能完全一致，0011.ai 提供的是**代理接入服务**。你使用的是相同的 Claude Code 能力，但通过 0011.ai 的 API 端点访问，解决了网络稳定性和支付问题。

### Q2: 封号风险如何？

**A**: 0011.ai 使用独立的 API 密钥体系，与官方账号体系隔离，**不存在封号风险**。

### Q3: 支持哪些编程语言？

**A**: Claude Code 原生支持 12+ 种语言和框架，包括 JavaScript/TypeScript、Python、Go、Rust、Java、C++、React、Swift 等。

### Q4: 如何切换回 OpenAI Codex？

**A**: 使用 0011.ai 客户端的用户，可在设置中一键切换；手动配置用户需修改 `ANTHROPIC_BASE_URL` 为 Codex 端点。

### Q5: 余额不足会怎样？

**A**: 余额不足时 API 调用会返回错误，建议开启余额提醒，或设置自动充值。
